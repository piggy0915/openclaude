# 田良良 重大决策与职业转折点

## 一、概述

田良良（1983年出生，河北邯郸人），毕业于中央戏剧学院制片管理专业（非编剧科班），后转型为职业编剧。她是灵河文化旗下签约编剧，曾任灵河十翼编剧工作室负责人。其职业生涯以《三体》电视剧总编剧为最高成就点，此前代表作品包括《麻辣女兵》《花千骨2015》《大话西游之爱你一万年》《终极笔记》等。

本文基于检索到的公开信息，系统梳理田良良职业生涯中的重大决策与转折点。

### 决策4：尝试20多版改编方案并最终选择"回归原著"（2016-2017年，历时约1-2年）

| 字段 | 内容 |
|------|------|
| **时间** | 2016年—2017年 |
| **背景** | 没有科幻剧改编经验可以参考，编剧团队陷入方向性迷茫。 |
| **决策内容** | 尝试近20种完全不同的改编方向，经过漫长的试错后，最终决定"回归原著"。 |
| **本人自述** | "第一次改编科幻作品，没有经验可以吸取。所以我们用了一个笨办法：把所有可以走的路都走一遍。做了二十多版方案，把能想象的都做了，最后决定回归原著。"（杭州日报，https://www.hzzx.gov.cn/cshz/content/2023-02/03/content_8462144.htm） |
| **关键细节** | 这些尝试方案包括：将时间线调整到2030年（加入无人驾驶出租车、先进VR设备等）；将杨冬以量子化状态呈现；走大众路线做成罪案题材等。但这些路径都走不通，"很多增加的桥段和刘慈欣小说的气质完全融不到一起"。 |
| **来源** | 杭州政协网（https://www.hzzx.gov.cn/cshz/content/2023-02/03/content_8462144.htm）；三联生活周刊（http://ny.slzhongdu.com/h5/article/detail.do?artId=191623） |
| **可信度** | 高（杭州日报/三联生活周刊均为正规媒体，内容经交叉验证） |
| **类型** | 本人所述 |

**决策过程还原（综合多个来源）**：
1. 田良良团队首先尝试了"大众化"路线，计划将《三体》改编为罪案题材以降低门槛，但发现这与刘慈欣小说的气质无法融合。
2. 尝试调整时间线至2030年以增强"科幻感"，但同样被否决。
3. 尝试各种增强戏剧冲突的手法，但都与原著的"克制"气质冲突。
4. 在多轮尝试无效后，田良良反复阅读原著数十遍，最终领悟到原著的独特气质——"中国人特有的克制"。
5. 在平台方、版权方及原著作者刘慈欣参与的创作会上，确定了核心方向："按照原著的脉络，并且用现实主义的手法来表达。"
6. 白一骢补充说："我们用了非常笨的方法，就是把所有可以走的路都走了一遍。"

**关于"克制"的原创性洞察**：
田良良在多次阅读原著后指出——"这部科幻作品的独特之处在于它的整体气质和氛围特别有中国人特有的克制，这种克制弥漫在整部小说里，它所表现的人与人之间的情感，抑或叶文洁对地球情感的变化，它所构建的科幻想象是内在的、激烈的，但外面裹着一层克制的现实外壳，这使得整个小说充满着特殊的张力。"（三联生活周刊）

## 四、面对困难与挑战的应对方式

### 4.1 面对"无经验可循"的困境

**应对方式**：采用"穷尽法"（试遍所有可能的选项）→ 通过排除法找到正确答案。田良良和白一骢称之为"笨办法"——把所有可以走的路都走一遍。

**评价**：这种应对方式体现了她的务实性格和不畏试错的勇气。"笨办法"的表面意思是谦逊，实质是一种系统化的决策方法论。

### 4.2 面对精神压力的应对

**应对方式**：团队封闭式协作（吃住在一起）+ 反复阅读原著寻找方向感。田良良承认"精神上的艰苦"远大于"生活上的艰苦"，但在迷茫中她选择了"更深入地回到作品本身"而非放弃。

**旁人视角**：白一骢说"灵河的编剧团队核心人员一直都在，他们坚持了很多年"。

**评价**：体现了高度的韧性和团队凝聚力。

### 4.3 面对原著粉丝期待的应对

**应对方式**：同名场面"做细致筹划"，找到名场面打动人心的核心原因，在此基础之上做"观众能接受的合理影视化表达"。田良良坦言"不敢看弹幕"，体现了对口碑的高度在意和创作压力。

### 4.4 面对科幻概念影视化难题的应对
