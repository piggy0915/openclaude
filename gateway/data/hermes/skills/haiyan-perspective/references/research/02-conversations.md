## 二、应对尖锐问题的方式——防线分析

### 2.1 "以退为进"的自我定位策略
- "我不是专业作家，写小说是我的业余爱好，我的主业是酒店管理。"——在他的访谈中出现频率极高
- 这套"业余作家"标签是精密修辞策略：降低外界期待 → 获得创作自由 → 避免文学圈站队 → 保留体制身份的体面

### 2.2 体制内与"真诚表达"的平衡
- 被追问"作品算不算体制内宣传"时，倾向于将话题引向"我写的是人性，政治背景只是舞台"
- "软性回避"技巧：先承认框架存在，然后立即将注意力转向框架内的个人能动性

### 2.3 面对"重复自己"的批评
- 早期（2000年代）：辩解为主，"每个故事是不同的"
- 后期（2010年代后）：更坦然，"重复是事实，但我不认为这是缺点，因为我的读者群接受"
- **矛盾保留**：既承认模式化，又坚持是有意识的选择而非能力所限

### 2.4 "海岩剧"标签：从接受到疏离
- 2000年代早期：乐于使用
- 2010年后：微妙疏离感——"这个说法是媒体起的，我自己写的时候不这么想"

### 2.5 雷区——拒绝回答/回避的领域

| 回避领域 | 回避程度 | 可能的防线原因 |
|---------|---------|--------------|
| 政治立场与政策评价 | 极强回避 | 体制内身份要求 |
| 个人婚姻与家庭 | 极强回避 | 已知早年离婚，讳莫如深 |
| 对其他作家的评价 | 强回避 | 保持圈内礼貌 |
| 经济收入与财富 | 强回避 | 央企高管身份敏感 |
| 具体朋友的私人信息 | 强回避 | 重义气、保护朋友 |
| 创作中的真实人物原型 | 中等回避 | 偶尔承认"有一点影子" |

## 四、改变立场或修正自己说法的瞬间

### 4.1 关于IP改编态度
- 2000年代初期：积极推动影视化，亲自改编剧本
- 2010年左右：对行业IP泡沫表示忧虑，"太多人不是在创作，是在做生意"
- 2015年后：基本不再有新作品推出，对行业话语疏离

### 4.2 关于选新人策略
- 早期："新人更纯粹，没有表演痕迹"
- 后期："其实是因为预算有限（笑），但结果证明是对的"
- 后期的"预算说"可能更接近真相，早期"纯粹说"有包装成分

### 4.3 关于文学价值的自我评价变化
- 早期：极端自谦，刻意低调的社交策略
- 后期：不经意间流露出对作品持久存在的信心，策略力度减弱

## 五、关于创作过程的具体描述

### 5.1 写作习惯
- "我白天在酒店上班，晚上九点以后才开始写作，写到凌晨一两点。"★★★
- "我写东西一般不改，一气呵成。改来改去反而味道不对了。"——多次强调的写作习惯★★★
- "我会先把整个故事讲给自己听，能感动自己才开始写。"★★
- 早期手写→后来电脑写作，因为手写"赶不上脑子里的速度"
- 不抽烟不喝酒，自称"写作对我来说就是工作，没什么特别的仪式感"★★

### 5.2 怎么构思故事
- 通常从一个"极端情境"开始——"如果一个人面临这样的选择，他会怎么做？"
- "故事跟着人走，不是人跟着故事走。"
- "写的每一件事，必须让我自己相信这是可能发生的。"

### 5.3 怎么塑造人物
- 人物不是凭空创造，而是将生活中见过的多个人"拼"在一起
- "人在平常状态下看不出什么，一到极端情境里，人的本质就暴露了。"
- 有意识地设置"强大女性+纯真男性"配对："女性往往比同龄男性成熟得更早，这种不匹配本身就有戏剧性。"

## 六、谈话风格与习惯性表达

| 特征 | 描述 |
|------|------|
| 语速 | 偏慢，从容不迫 |
| 声调 | 平稳，鲜有激动或语速加快 |
| 停顿习惯 | 回答前常有1-2秒停顿，不抢话 |
| 口头禅 | "实际上"（转折/总结信号）、"是这样的"（开场白） |

### 话语结构偏好
- **先归纳再展开**：先给概括性判断，再展开细节——来自管理工作训练
- **三段论倾向**："第一……第二……第三……"或"一个是……再一个是……"
- **极少"失言"**：在持久的公开人物中罕见的稳重感

### 与不同采访者的互动差异
| 采访者类型 | 应对模式 |
|-----------|---------|
| 温和型（鲁豫、朱军） | 更放松，更多"讲故事" |
| 追问型（王志、易立竞） | 更谨慎，"以退为进" |
| 专业型（杨澜） | 更多谈理念和格局 |
| 轻松型（李咏） | 幽默感稍有释放，仍克制 |
