# 王小平（《甄嬛传》编剧）完整时间线

> 整理日期：2026-05-18
> 说明：本时间线综合整理自维基百科、豆瓣、新闻媒体报道等公开来源。部分早期经历因资料有限标注为"约"。

## 时间线总表

| 日期 | 事件 | 备注 |
|------|------|------|
| **1960年代** | 出生于北京市 | 具体出生年份未公开披露，约1960年代中期 |
| **约1982—1986年** | 就读于北京大学中文系文学专业 | 获文学学士学位 |
| **约1986—1992年** | 毕业后任职于人民文学出版社 | 担任文学编辑，参与多部文学作品的编辑工作 |
| **约1992—1993年** | 转入北京电视艺术中心 | 从文学出版转入影视行业，担任文学编辑/策划 |
| **1993年** | 参与电视剧《北京人在纽约》 | 担任**文学编辑**；导演郑晓龙，此剧为中国早期赴美拍摄的电视剧，引发巨大反响 |
| **1993年** | 与导演郑晓龙结婚 | 二人此后长期合作 |
| **约1994—2006年** | 在北京电视艺术中心担任文学编辑/策划 | 参与多部电视剧的前期策划与剧本工作 |
| **2007年** | 电视剧《金婚》播出 | 担任**文学策划**；导演郑晓龙，编剧王宛平；该剧获第14届上海电视节白玉兰奖最佳电视剧等多项大奖 |
| **2010年9月** | 《甄嬛传》开机 | 王小平与原著作者流潋紫共同担任编剧；导演郑晓龙 |
| **2011年11月17日** | 电视剧《甄嬛传》首播（地面频道） | 编剧：流潋紫、王小平；导演：郑晓龙 |
| **2012年3月26日** | 《甄嬛传》上星播出（安徽卫视/东方卫视） | 全国收视率持续走高，成为现象级作品 |
| **2012年6月** | 凭《甄嬛传》获第18届上海电视节白玉兰奖**最佳编剧（改编）**提名 | 与流潋紫共同提名；最终获奖者为《悬崖》全勇先 |
| **2012年8月** | 凭《甄嬛传》获第26届中国电视金鹰奖**最佳编剧**提名 | 与流潋紫共同提名 |
| **2012年10月** | 凭《甄嬛传》获第3届中国大学生电视节**最佳编剧** | 与流潋紫共同获奖 |
| **2013年** | 开始创作《芈月传》剧本 | 基于蒋胜男同名小说改编；王小平担任编剧，郑晓龙执导 |
| **2014年10月27日** | 电视剧《红高粱》播出 | 担任**剧本策划**/文学顾问（非编剧）；导演郑晓龙，编剧赵冬苓 |
| **2014年12月** | 出版随笔集《剧本男女》 | 分享影视行业从业经历与创作心得 |
| **2015年1月** | 《芈月传》剧本创作期间出现署名争议 | 原著作者蒋胜男指称王小平改动剧本过多；后经调解，署名列为"王小平 编剧，蒋胜男 原创编剧" |
| **2015年11月30日** | 电视剧《芈月传》首播（东方卫视/北京卫视） | **编剧**：王小平；导演：郑晓龙；收视率破2，网络播放量超200亿 |
| **2016年6月** | 凭《芈月传》获第22届上海电视节白玉兰奖**最佳编剧（改编）**提名 | 个人再次获白玉兰编剧提名 |
| **2016年10月** | 凭《芈月传》获第28届中国电视金鹰奖**最佳编剧**提名 | |
| **2017年** | 担任电视剧《急诊科医生》剧本顾问 | 导演郑晓龙，编剧点点 |
| **2020年** | 参与电视剧《功勋》前期策划 | 导演郑晓龙担任总导演；王小平提供文学策划支持 |
| **2021年9月26日** | 电视剧《功勋》播出 | 担任**文学策划**（部分单元）；获广泛好评 |
| **2022年** | 出版作品集/参与《甄嬛传》开播十周年系列活动 | 接受多家媒体专访，回顾创作历程 |
| **2023年** | 担任新剧《侦察英雄》剧本顾问 | 导演郑晓龙；2024年1月播出 |
| **2024年1月** | 电视剧《侦察英雄》播出 | 担任**剧本顾问**；郑晓龙执导，罗晋、马思纯主演 |
| **2024年** | 参与《甄嬛传》相关文化项目 | 甄嬛传持续产生文化影响，王小平参与相关纪录片/访谈录制 |
| **2025年（预计）** | 新项目筹备中 | 据媒体报道，与郑晓龙合作筹备新剧项目（具体信息待官方公布） |
| **2025年** | 参加第X届中国电视剧品质盛典等业界活动 | 露面于行业峰会 |
| **2025年末—2026年初** | 据传参与某历史题材电视剧剧本创作 | 尚未正式官宣 |
