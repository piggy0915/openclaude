## 使用优先级

1. **首选**：平台官方榜单(红果/抖音App内 → WebSearch搜索)
2. **次选**：第三方数据平台(新榜/飞瓜 → WebSearch搜索)
3. **兜底**：行业媒体分析(36氪/晚点/虎嗅 → WebSearch搜索)
4. **参考**：知识库快照(SKILL.md内联数据 + references/research/)

> 搜索时优先使用中文query搜国内数据，英文query搜海外数据。如中文结果不足，可尝试用中文平台名+英文关键词交叉搜索。
