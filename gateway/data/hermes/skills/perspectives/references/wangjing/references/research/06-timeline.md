# 王晶 · 时间线

## 人物基本信息

- **本名**: 王日祥 (Wong Yat-cheong)
- **艺名**: 王晶 (Wong Jing)，曾用笔名「不是女人」「不是野人」
- **出生**: 1955年5月3日，香港
- **祖籍**: 浙江省绍兴市
- **教育**: 毕业于香港中文大学联合书院中文系（1978届），中学就读于香港培正中学
- **职业**: 导演/编剧/监制/制片人/演员
- **父亲**: 王天林（Wong Tin-lam，著名导演，香港电影先驱之一）
- **子女**: 王佟艺、王子涵（均为女儿）
- **马主**: 香港赛马会会员，曾拥有名下赛马

## 时间线

| 时期 | 阶段 | 关键事件/代表作 |
|------|------|---------------|
| 1955-1975 | 成长与教育 | 1955年5月3日出生于香港，父亲王天林是邵氏及TVB著名导演。中学就读培正中学，后在香港中文大学中文系就读。大学期间已开始为父亲担任场记等工作。1975年毕业，大学期间曾为父亲做电影日志记录（log keeper）。 |
| 1975-1980 | TVB入行期 | 1975年进入TVB担任编剧。1976年担任爱情剧《心有千千结》编剧，正式开启编剧生涯；同年担任武侠剧《陆小凤》编剧。1979年参与《楚留香传奇》《网中人》等经典TVB剧集编剧。1980年担任《京华春梦》《千王之王》编剧（后者开启赌剧先河），同年参与《亲情》（周润发主演）。此期间积累了丰富的剧本创作经验。 |
| 1981-1986 | 电影起步期 | 1981年执导首部电影《千王斗千霸》（邵氏出品），开启导演生涯。1982-1984年执导《贼王之王》《猎魔者》《花心大少》《青蛙王子》《我爱罗兰度》等多部商业片。1985年执导《鬼马飞人》《摩登仙履奇缘》。1986年执导刘德华主演的《魔翡翠》，成立个人制作公司雏形。此时以轻喜剧、爱情喜剧为主，尚未形成鲜明风格。 |
| 1987-1989 | 崛起与赌片开创 | **突破年**：1987年《精装追女仔》香港票房超2000万港元，奠定商业片地位。1988年推出《精装追女仔2》《最佳损友》系列，票房持续走高。1989年连推《至尊无上》《赌神》两部赌片，其中《赌神》以3706万港元夺得1989年香港年度票房冠军，周润发饰演的「赌神高进」成为影史经典。王晶由此开创香港赌片潮流。同年还执导了《专钓大鳄》《神勇双妹唛》等。 |
| 1990-1993 | 巅峰期·与周星驰合作 | 1990年与周星驰合作《赌侠》（年度票房亚军，4000万+港元），开启黄金搭档期。1991年《整蛊专家》《赌侠2：上海滩赌圣》《与龙共舞》等密集上映。同年担任制片人的《逃学威龙》《五亿探长雷洛传》均大卖。1992年执导周星驰版《鹿鼎记》《鹿鼎记Ⅱ：神龙教》，同年推出《赌城大亨之新哥传奇》（何鸿燊传记）。1993年产量惊人：执导《逃学威龙3》《城市猎人》《黄飞鸿之铁鸡斗蜈蚣》《超级学校霸王》《倚天屠龙记之魔教教主》等近10部电影，并监制多部作品。此期间与周星驰、刘德华、李连杰、周润发、张敏、邱淑贞等大量一线演员频繁合作。 |
| 1994-1996 | 巅峰持续与新系列 | 1994年连续推出《新少林五祖》《九品芝麻官之白面包青天》（周星驰主演，豆瓣9.3分神作）、《赌神2》（香港票房5250万港元，创王晶个人香港最高票房纪录）、《香港沦陷》等。1995年执导《鼠胆龙威》《赌圣2：街头赌圣》等。1996年开辟「古惑仔」系列（以监制/制片人身份推出《古惑仔之人在江湖》），监制《大内密探零零发》，执导《赌神3之少年赌神》。古惑仔系列成为又一卖座IP。 |
| 1997-2000 | 转型与公司运营 | 1997年与文隽等成立「最佳拍档电影公司」。1998年《龙在江湖》《赌侠1999》等。1999年《千王之王2000》（周星驰客串）、《笨小孩》等。2000年《赌圣3无名小子》《中华赌侠》等。港片市场整体下滑，王晶产量减少但仍保持年均多部。 | 
| 2001-2009 | 北上试水与剧集双线 | 2001年执导《情迷大话王》（黎明、张柏芝）、《我的野蛮同学》等。2003年《黑白森林》《神勇铁金刚》。2004年《新扎师兄》《精装追女仔2004》。2005年执导内地武侠剧《小鱼儿与花无缺》（谢霆锋、张卫健主演，年度热门剧集），编剧/制片《天下第一》（邓衍成导演，李亚鹏、霍建华主演），同年凭《黑白战场》获第12届香港电影评论学会大奖**最佳编剧奖**。2006年凭《卧虎》再获香港电影评论学会大奖**最佳编剧奖**。2007-2009年间执导多部内地电视剧（《A计划》《雪山飞狐》《仁者黄飞鸿》），兼任多部电影监制（包括许鞍华《天水围的日与夜》《天水围的夜与雾》），展现了对艺术片的支持。 |
| 2010-2016 | 北上成熟期·澳门风云系列 | 2010年《未来警察》（刘德华）、《美丽密令》等。2011年《财神客栈》《无价之宝》。2014年《澳门风云》内地票房2.67亿人民币，开启北上后最成功系列。2015年《澳门风云2》内地票房9.75亿人民币。2016年《澳门风云3》内地票房11.18亿人民币，系列累计超23亿。此间也尝试严肃题材，2017年执导《追龙》（刘德华、甄子丹主演），豆瓣7.2分，口碑为北上后最佳，被视为「王晶认真起来也能拍好片」的代表作。 |

| 2024-2026 | 最新动态 | 2024年参演《红毯先生》（宁浩导演），参加综艺《无限超越班第二季》《小闯横莱坞》。监制网络电影《怒火12小时》。监制《黑金对决》开机。2025年1月监制《笑傲江湖》（张雨绮饰东方不败）上线爱奇艺/腾讯/优酷，香港首日票房仅1.2万港币，口碑差评为主；同日执导的港片《麻雀女王追男仔》香港上映。3月参加《无限超越班第三季》。8月出版自传《人生由我掌控》，在广州南国书香节首发。12月在社交平台评论张国荣/梅艳芳/陈百强引发争议。获香港电影编剧家协会**荣誉大奖**。2026年编剧/监制电影《解决专家》上映。YouTube频道「王晶笑看江湖」持续更新（EP92-93等），回顾50年影视生涯。 |

## 风格/思想转折点

1. **1976年TVB入行**：从中文系毕业生转变为职业编剧，师从父亲王天林，习得"观众至上"的商业叙事逻辑。
2. **1989年《赌神》**：从轻喜剧转向赌片+英雄类型，开创「赌神宇宙」，确立商业片"类型杂交"方法论（赌术+喜剧+兄弟情）。
3. **1990-1992年与周星驰合作**：将无厘头喜剧与赌片深度结合，创作密度最高时期（年均5-10部），培养了极度高效的"快拍快上"制片模式。
4. **1996年古惑仔系列**：从导演转型为"制片人/IP操盘手"，证明自己不止是导演，更是商业嗅觉敏锐的电影企业家。
5. **2005年凭《黑白战场》《卧虎》获最佳编剧奖**：证明自己并非只会拍"屎尿屁"商业片，具备严肃叙事能力。
6. **2014年《澳门风云》开启北上新时代**：全面转向内地市场，利用怀旧牌+明星堆砌策略，票房大获成功但口碑两极。
7. **2017年《追龙》**：被视为"认真王晶"的能力证明，验证了严肃历史题材的驾驭力，但随后《追龙2》口碑回落，显示风格缺乏连续性。
8. **2022年《新倚天屠龙记》网络上映**：传统院线思维向流媒体转型的标志性节点，但口碑崩塌暴露了经典IP消费化策略的局限。
9. **2025年出版自传+获编剧荣誉大奖**：从创作者转向"历史书写者"角色，在行业地位获得认可的同时，最新作品口碑持续下滑。

## 关键市场数据

- **导演/编剧/监制作品总数**：超过400部影视作品，执导约60余部电影
- **香港年度票房冠军次数**：1次（1989年《赌神》）
- **个人香港最高票房作品**：《赌神2》（5250万港元）
- **内地最高票房作品**：《澳门风云3》（11.18亿人民币）
- **豆瓣最高评分作品**：《九品芝麻官》（9.3分）
- **豆瓣最低评分作品**：多部低于4分（如《唐伯虎冲上云霄》等）
- **获得重要编剧奖项**：2次香港电影评论学会大奖最佳编剧（《黑白战场》《卧虎》），1次香港电影编剧家协会荣誉大奖（2025）
- **金扫帚奖**：曾获最烂导演奖项
