## 一、关键职业转折点

### 1.1 从设计师转行编剧（约2001年）

| 维度 | 内容 |
|------|------|
| **背景** | 王倦本为设计师，通过组里导演推荐给于荣光 |
| **决策** | 受邀编剧《侦探成旭》系列，从情景喜剧转型长篇电视剧编剧 |
| **王倦自述** | 🧑"先看到画面再写台词"——设计背景塑造了他的编剧方式 |

### 1.2 加入"荣光教"（约2007年起）

| 维度 | 内容 |
|------|------|
| **背景** | 蒋晓荣（制片人）+ 于荣光（导演）+ 王倦（编剧）的铁三角，最终合作10次 |
| **决策模式** | 编剧中心制的早期实践——王倦负责剧本核心，于荣光提供制作资源和表演保障 |
| **输出期** | "边陲三部曲"——《木府风云》(2012, 8.3)、《舞乐传奇》(2013, 8.7)、《南侨机工英雄传》(2015, 8.2) |
| **可信度** | 🟢 高（搜狐2019行业分析 + 凤凰网2023深度报道） |
| **意义** | 这是王倦创作自由度最高的时期——原创剧本、编剧中心、口碑优秀 |

### 1.3 原创→IP改编的转向（约2016年）

| 维度 | 内容 |
|------|------|
| **背景** | 关键人物于婉琴（原云南电视台节目购销中心主任→新丽传媒高级副总裁）牵线，王倦签约新丽，"荣光教"逐渐解散 |
| **王倦自述** | 🧑（未直接回应是否自愿转向）从行动轨迹看，签约新丽后全部项目皆为改编 |
| **外部推测** | 👥《博客天下》(2023.10)分析认为市场压力是主因——"IP+流量模式下原创剧本难以获得平台认可" |

| **⚠️矛盾** | 王倦从未公开说"我不想做原创了"，但在签约新丽后的7年里，他确实几乎只做改编 |

**意义**：这是理解王倦职业生涯最关键的转折点——他的创作身份从"原作者"变成了"改编者"。

## 二、改编决策案例

### 2.1 滕梓荆：从奴仆到兄弟（《庆余年》）

| 维度 | 内容 |
|------|------|
| **原著设定** | 滕梓荆为范闲的奴仆/护卫，在故事中并非关键角色 |
| **王倦改动** | **完全重写**——改为范闲的好友/守护者，增加大量戏份和情感联结 |
| **王倦解释** | 🧑"需要在前期有一个角色让范闲的复仇有情感分量" |
| **效果** | 滕梓荆之死成为《庆余年》第一季最催泪的段落之一，弹幕"刀死我了" |
| **决策模式信号** | 为了**情感效果**可以**大幅重构**角色——哪怕得罪原著粉 |

### 2.2 范闲"去黑暗面"（《庆余年》）

| 维度 | 内容 |
|------|------|
| **原著定位** | 范闲杀伐果断、对生命有某种漠视 |
| **王倦改动** | 去掉黑暗面，让范闲"更善良、更可爱" |
| **王倦解释** | 🧑"主线故事比较沉重，希望范闲身上能让观众感受到人性的温暖" |
| **争议** | 原著读者批评角色复杂度被削弱；剧粉认为更容易共情 |
| **决策信号** | 王倦在"大众接受度"和"原著忠实度"之间的权衡，**他总是优先选择前者** |

### 2.3 王启年：内心独白外化

| 维度 | 内容 |
|------|------|
| **创作问题** | 小说可以大段心理活动，但电视剧不能 |
| **王倦解法** | 创造王启年这个"陪伴性角色"，把范闲的内心独白外化成两人对话 |
| **决策本质** | 从**媒介特性**出发做改编——"小说适合做的，电视剧不一定适合" |

### 2.4 分季逻辑与节奏控制（《庆余年》）

| 维度 | 内容 |
|------|------|
| **决策** | 《庆余年》分成三季，每季有完整的人物弧 |
| **逻辑** | "每次改编完后，这个季的高潮落在哪里，人物有什么成长" |
| **最新** | 🧑2024年5月确认"第三季不会等这么久"，2025年6月官宣第三季剧本完成约80% |
| **可信度** | 🟢 高（广州日报2024.05 + 官方报道） |

### 2.5 《斗罗大陆》改编争议

| 维度 | 内容 |
|------|------|
| **改编策略** | "砍"掉大量细枝末节，专注主线成长 |
| **王倦回应** | 🧑公开发微博回应争议，解释玄幻视觉化困境——"蓝银草"要怎么做才好看？ |

### 2.6 《大宋少年志》系列

| 维度 | 内容 |
|------|------|

## 三、决策模式分析

### 3.1 选项目的标准

基于项目选择模式，王倦的核心考量排序：

| 优先级 | 考量 | 证据 |
|--------|------|------|
| 1 | **IP质量** | 猫腻、烽火戏诸侯、唐家三少都属头部网文作者 |
| 2 | **创作空间** | 是否允许他做结构性调整？从"搭积木""重写角色"可见他要空间 |
| 3 | **团队信任** | 从"荣光教"到新丽体系，"信任"是其合作的基石 |
| 4 | **类型切换** | 从古装到谍战（《千里江山图》），可见他不愿被单一类型框死 |

### 3.2 批评回应模式

王倦面对争议有一套稳定的回应模式：
1. **被动回应**：一般不主动挑起讨论，等争议发酵才回应
2. **坦诚优先**：愿意承认"有失误"——这在编剧群体中少见
3. **解释不辩解**：解释创作逻辑，但不过度辩护
4. **公开渠道**：主要使用微博（他的主要公众平台）

### 3.3 合作选择逻辑

王倦的职业生涯体现了**"跟对人"**的智慧：
- 早期：绑定于荣光（10次合作）
- 中期：通过于婉琴进入新丽体系
- 持续：与张若昀多次合作（庆余年+千里江山图）
- 模式：不靠"广撒网"而是"深度绑定"——与核心合作者反复合作

## 四、核心决策模式总结

### 4.1 五个特征

1. **保守前进型**——不轻易跳脱安全区，但在安全区内逐步扩展边界
2. **结构优先型**——决策时关注的是**结构问题**（节奏、高潮、人物弧），而非细节
3. **情感导向型**——在"忠实原著"和"情感效果"之间，**每次都选情感效果**
4. **风险回避型**——回避政治表态、回避评价同行、回避制造话题
5. **被动成长型**——外部环境推动的转型多于主动求变

### 4.2 决策驱动力权重

| 驱动因素 | 权重 | 证据 |
|---------|------|------|
| 创作自由 | 高 | "荣光教"时期最有创造力 |
| 团队信任 | 高 | 于荣光10次合作、新丽长期绑定 |
| 市场接受 | 中高 | 重视观众反应（但也说"不为受众创作"） |
| 个人表达 | 中 | 在改编中嵌入个人价值观（如"势均力敌"的女主） |
| 经济利益 | 低 | 公开资料缺乏证据，无法判断 |

### 4.3 关键弱点

- 过于依赖"反转"套路——尚未找到有效的进化路径
- 转型是被动的——从原创到改编的转向是被行业裹挟，而非主动选择
- 舒适区明显——古装+权谋+喜剧高度重合，类型突破要等《千里江山图》验证
